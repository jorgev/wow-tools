Introduction

This site can be used to parse World of Warcraft combat log files. Combat data is logged any time you enable combat logging by using the /combatlog option in-game. The data is written to a file named WoWCombatLog.txt in Logs folder of your installation directory. Combat logging is not enable by default, you must enable it using the option mentioned. Furthermore, the option to log is not persisted, so you must re-enable anytime you log out or switch characters and come back into the game.

How Does It Work?

This parser works by allowing you to upload your combat log file. Keep in mind that combat files can get very large, so you may want to break up individual encounters into smaller files. Also, the combat log does not log entering/leaving combat events, so it is difficult to get precise DPS/HPS calculations. Whenever possible, you'll want to trim your combat log files to only include the time range you are specifically interested in. Generally, the best way to do this is to look for the very first and last entries for a particular encounter, deleting everything before the first and after the last event. That way, the calculated stats should be fairly accurate. Results are displayed on a web page with charts.
